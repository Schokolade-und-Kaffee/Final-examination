關於我們

物理三甲_許玉綸

我最大的收穫莫過於學會使用 Git 和 Github 了， 以前也是一昧地覺得就是程式的雲端硬碟。 雖然一路上來非常地艱苦，但好歹也算是學會一項技能。 我認為最困難的就是搞清處本機端與遠端所謂的「分支（branch）」之於「儲存庫（repository）」的概念了。 其次則為理解常用指令字元背後的涵義，往往想達到某種效果卻有不同的寫法，對初學者來說有些困難（資訊爆炸），而我熬過來了。說回這次的專案，我還是會覺得見識是最重要的。有一種說法是說「人沒辦法想像到沒看過的東西。」我很高興之前有跟其他資管的同學在資訊管理的期末一起製作專案， 雖然當時的我幫不太上忙，但那時觀摩的經驗確實地成為現在眼前這個專案的養份。還有深深地體悟到共同開發專案時，溝通成本究竟有多大。尤其是在上了大半學期的軟體專案管理後。
